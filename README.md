# TradeHub UI

Базовый интерфейс универсальной торговой площадки:

- `index.html` — публичная главная страница;
- `dashboard.html` — состояние интерфейса после входа;
- `styles.css` — общие стили для обеих страниц.

## Локальный запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 4173
```
